# oratorIA

Uma aplicação web para treinar argumentação através de debates com IA.

## 🚀 Funcionalidades

-   Escolha entre diferentes temas de debate
-   Selecione o nível de dificuldade (fácil, médio, difícil)
-   Receba respostas críticas e educadas da IA
-   Interface simples e intuitiva

## 🛠️ Tecnologias

-   Next.js 14
-   TypeScript
-   Tailwind CSS
-   Google Gemini AI API

## 📋 Pré-requisitos

-   Node.js 18+ instalado
-   Uma chave de API do Google Gemini AI

## 🔧 Configuração

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/oratoria.git
cd oratoria
```

2. Instale as dependências:

```bash
npm install
```

3. Crie um arquivo `.env.local` na raiz do projeto e adicione sua chave da API Gemini:

```
GEMINI_API_KEY=sua_chave_api_aqui
```

4. Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

5. Acesse a aplicação em `http://localhost:3000`

## 📝 Como Usar

1. Selecione um tema de debate
2. Escolha o nível de dificuldade
3. Digite seu argumento
4. Clique em "Enviar Argumento"
5. Leia a resposta da IA e continue o debate
